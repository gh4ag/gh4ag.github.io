FreeFem++ class examples

FreeFem++ is a finite-element (and finite volume) solver written in C++ with adaptive meshing. See the official FreeFem++ page for more information about the software.

My contribution to this project consist in some short examples to show how to embed a user-defined C++ class into FreeFem++ using the right FreeFem++ functions and classes.

Link to the github repo: freefem_class_examples
